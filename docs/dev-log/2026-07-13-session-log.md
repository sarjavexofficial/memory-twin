## バージョン履歴

### 0.1.6（2026-07-13）自分のタスク（期限つき）
ゆずの提案「AIに自発的に確認させるのではなく、ユーザーが◯月◯日までにこれ、と決められた方がいい」への回答。
AIが記録から拾う約束は残しつつ（差別化点のため）、**ユーザーが期限を決めて登録するタスク**を追加した。
- 新データ層 `src/lib/task-data.ts` + `src/store/tasks-context.tsx`（保存キー `memory-twin:tasks`、破損時はstorage-guardで退避）
- Today Recallの優先順位を「①自分のタスク（期限が近い順）→ ②AIが拾った約束 → ③誕生日」に変更。期限バッジは約束と共通
- Recall欄は空でも常時表示し、「＋ 期限つきタスクを追加」から追加（やること・いつまでに・関係する人[任意]）
- 完了はタップ→取り消しバナー、削除は行の長押し→取り消しバナー（誤操作はどちらも戻せる）
- タスク1件でも「本人のデータあり」扱いにし、サンプル（デモ）データをRecall・ひとこと・今年のあなたから退場させる
  （当日固定済みのひとことも、サンプル人物が話題なら実データ登場時点で非表示にするよう dailyCandidateValid を拡張）
- 「今年のあなた」の果たした約束に、今年完了したタスクを「自分との約束」として合算
- バックアップ完全対応: ZIPエクスポート（data.json+HTML）・暗号化クラウド・復元・全削除にtasksを追加（旧バックアップとの互換維持）
- 6言語対応。E2E: タスク26項目＋回帰16項目 全緑
